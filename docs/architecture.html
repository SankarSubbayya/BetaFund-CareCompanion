3-Agent Flow

sequenceDiagram
    participant OA as 🤖 Onboarding Agent
    participant DA as 🤖 Daily Check-in Agent
    participant SA as 🤖 Scheduling Agent
    participant EM as 🧠 EverMind
    participant B as 🎙️ Bland AI
    participant Sr as 👴 Senior
    participant Fam as 👨‍👩‍👧 Family

    Note over OA: Agent 1: Onboarding
    Fam->>OA: Register loved one
    OA->>B: Voice onboarding call
    B->>Fam: Collect name, phone, meds, schedule
    B-->>OA: Transcript
    OA->>OA: Extract info + create profile

    Note over DA: Agent 2: Daily Check-in
    SA->>DA: Daily cron trigger
    DA->>EM: Recall past conversations
    EM-->>DA: "Hip hurting, missed meds last time"
    DA->>B: Call with memory context
    B->>Sr: 📞 "Hi Margaret, how is your hip?"
    Sr-->>B: Voice conversation
    B-->>DA: Transcript

    DA->>DA: Analyze mood, meds, services
    DA->>EM: Store new memory
    Note over EM: "Margaret feeling better,
took meds, needs groceries"

    alt Service request detected
        DA->>DA: Match local SF providers
    end

    Note over SA: Agent 3: Scheduling & Notification
    alt Critical/High alert
        DA->>SA: Alert detected!
        SA->>B: Notification call
        B->>Fam: 📞 "Margaret fell, needs attention"
    end

    Fam->>Fam: View dashboard
    Note over Fam: Real-time alerts,
wellness trends,
service recommendations

Data Model

erDiagram
    SENIOR {
        string phone PK "E.164 format"
        string name
        list medications
        list emergency_contacts
        string checkin_schedule "HH:MM"
        string notes
    }

    CHECKIN {
        string key PK "phone:timestamp"
        string senior_phone FK
        string call_id
        string timestamp
        string transcript
        string mood "happy|neutral|sad|concerning"
        int wellness_score "1-10"
        bool medication_taken
        list concerns
        list service_requests
        string summary
    }

    ALERT {
        string id PK "phone:timestamp:type"
        string senior_phone FK
        string senior_name
        string timestamp
        string alert_type "emergency|low_mood|missed_med|service_request"
        string severity "critical|high|medium|low"
        string message
        bool acknowledged
    }

    MEMORY {
        string message_id PK
        string sender "senior phone"
        string content "conversation summary"
        string create_time
        string memory_type "episodic_memory"
    }

    SENIOR ||--o{ CHECKIN : "has many"
    SENIOR ||--o{ ALERT : "triggers"
    CHECKIN ||--o{ ALERT : "generates"
    SENIOR ||--o{ MEMORY : "remembered by EverMind"
